# IOE Notes Hub

A centralized platform for IOE students to access, share, and manage academic resources based on the latest IOE syllabus.

## Project Goal

To create a reliable, user-friendly, and sustainable academic resource repository for IOE students.

## Problem Statement

IOE students often struggle to find organized study materials because notes and resources are scattered across social media groups, messaging platforms, and personal drives. Additionally, students face difficulties finding notes, resources, and syllabus information aligned with the new syllabus (2080).

## Solution

IOE Notes Hub provides a centralized platform where students can browse, upload, and download notes organized by semester and subject according to the new IOE syllabus (2080).

## Key Features

* Semester-wise organization
* Subject-wise categorization
* Search functionality
* PDF preview and download
* Note upload system
* Admin approval and moderation
* User authentication
